JavaScript :


  • JavaScript is an interpreted language, which means that code is executed line by line as it is encountered.
  • JavaScript is primarily used for client-side scripting within web browsers.
  • JavaScript is supported by all major web browsers, including Chrome, Firefox, Safari, Edge, and Opera.
  • JavaScript is an object-oriented programming language, which means it supports the creation and manipulation of objects.
  • JavaScript is dynamically typed, meaning that variables are not bound to a specific data type during declaration.
  • JavaScript is subject to security vulnerabilities such as Cross-Site Scripting (XSS) and Cross-Site Request Forgery (CSRF).
  • JavaScript can be used for a myriad of different applications.
  • JavaScript is both an imperative and declarative type of language.
  • First appeared: 1995.
