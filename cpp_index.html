C++ :


  • C++ is a cross-platform language that can be used to create high-performance applications.
  • C++ was developed by Bjarne Stroustrup, as an extension to the C language.
  • C++ gives programmers a high level of control over system resources and memory.
  • C++ was initially known as “C with classes, ” and was renamed C++ in 1983.
  • The first C with compiler was referred as Cfront, that derived from a C compiler referred to as CPre.
  • Object-Oriented Programming and Dynamic Memory Allocation.
  • First appeared: 1985.
  • OS: Windows, Linux, and Mac OS.
  • Paradigm: Procedural, Object-oriented and Generic programming
